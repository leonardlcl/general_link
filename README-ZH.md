# GeneralLink

## 安装

下载 `custom_components/general-link` 目录并将其复制到 HomeAssistant 配置文件夹中的 `custom_components` 目录

或者您可以使用 [HACS](https://hacs.xyz) 安装组件

## 配置
### step1
通过APP添加网关和其他智能设备
### step2
依次选择 【配置】 > 【集成】 > 【➕ 添加集成】

选择或者输入并选择【GeneralLink】
### step3
选择智能灯光控制类型：单灯/灯组，点击 【提交】

单灯：按照单个设备控制智能灯光

灯组：按照房间和灯组控制智能灯光
### step4
选择一个已扫描到的网关，点击 【提交】

完成后即可将设备同步到HomeAssistant
